California COVID-19 Vaccination Data

The State of California currently publishes COVID-19 vaccination data for the state as a whole and for each county, but is not in a useful, machine-readable format and it is only a current, point-in-time snapshot, with no history. This site hosts data files that are built by scraping the state website once a day.

  • A current snapshot of vaccinations is at current.v1.json
  • A timeseries of vaccinations in newline-delimited JSON format is at timeseries.v1.json
